# Prasad Khandale Portfolio (GitHub Pages Ready)

This folder contains a **static portfolio website** that you can directly host on **GitHub Pages**.

## Files
- `index.html` → main website page
- `styles.css` → all design and responsive styling
- `script.js` → animations, dark/light mode, counters, mobile menu

## How to host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Save.
6. Wait 1–2 minutes. Your site will be live at:
   - `https://yourusername.github.io/repository-name/`

## Important edits you should make
Open `index.html` and replace these placeholders:
- `your-email@example.com`
- `#` for LinkedIn link
